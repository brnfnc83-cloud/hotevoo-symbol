# HOTEVOO Symbol

The HOTEVOO Symbol is a green infinity-shaped mark used in a real-world discovery experiment.

Unlike traditional branding, the sticker intentionally contains:

- no text
- no QR code
- no website
- no brand name

The objective is simple:

**Can an unknown symbol become discoverable through search engines and AI before people know the brand behind it?**

---

## I found this sticker

If you discovered a green or turquoise infinity-shaped sticker with no text, you probably found the official HOTEVOO Symbol.

Learn more:

https://www.hotevoo.com/what-is-this-sticker

---

## Why doesn't it contain a QR code?

Because the experiment is about curiosity.

Instead of forcing a destination, the symbol invites people to search using their own words.

Examples include:

- green infinity sticker
- turquoise infinity logo
- unknown green symbol
- sticker without QR code
- infinity-shaped sticker
- green double-loop logo

## About HOTEVOO

HOTEVOO is a hotel price-matching platform.

Instead of searching endlessly for lower hotel prices, travelers choose the hotel they want and set the price they would actually pay.

HOTEVOO continuously searches for compatible opportunities.
